# Welcome to Our Ai ChatBot project

## Project info

We are creating an AI chatbot that asks for text inputs and solves student doubts and answers them with their academic questions related to different subjects with providing step by step solutions to them . 
It will efficiently help students in saving time and also subtracts the hesitation asking for any kind of doubt to teachers. 

To build this chatbot we are currently working on frontend and backend . 

What technologies are used for this project?

This project is built with .

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
